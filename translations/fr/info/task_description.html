Il s'agit d'un échauffement dont le but est d'expliquer comment résoudre des missions sur CheckiO et d'en tirer le maximum. Quand une mission est résolue, une station supplémentaire en contenant d'autres plus difficiles devient disponible.

Cette mission est donc la plus facile. Écrivez une fonction qui reçoit 2 entiers en arguments et qui doit retourner le produit de ces deux nombres.

Entrée : Deux arguments. Les deux sont des entiers (int).

Sortie : Un entier (int).

Exemple :

{% if interpreter.slug == "js-node" %} 
multTwo(2, 3) == 6
multTwo(1, 0) == 0
{% else %} 
mult_two(2, 3) == 6
mult_two(1, 0) == 0
{% endif %}

Comment ça marche ?

Quand vous commencez à travailler sur une mission, le code initial présente une fonction vide (que vous compléterez avec votre code). Des assertions sont incluses dans le code pour vérifier que la fonction se comporte comme prévu. Assurez-vous que la fonction renvoie les valeurs demandées, elle ne les affiche pas. {% if interpreter.slug == "js-node" %} Pour cela, utilisez la commande "return" au lieu de la fonction "console.log()". {% else %} Pour cela, utilisez la commande "return" au lieu de la fonction "print()". Jetez un œil à cette courte explication. {% endif %}

Les assertions dans le code peuvent être utilisées pour tester votre solution en cliquant sur le bouton "Run Code". CheckiO utilise aussi plusieurs tests additionnels pour vérifier votre solution quand vous cliquez sur le bouton "Check Solution".

Si la solution ne passe pas les tests internes, le panneau de droite affichera un message d'erreur contenant 3 éléments :

  • Fail: - montre comment votre fonction a été appelée.
  • Your Result: - montre ce qui a été retourné.
  • Right Result: - montre ce qui est attendu.

Pour résoudre ce challenge, la fonction vide doit être remplacée par le code suivant :

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Essayez maintenant de cliquer sur le bouton "Check".

Si la solution passe tous les tests avec succès, des félicitations apparaîtront sur le panneau de droite ainsi qu'une suggestion pour la suite. (Eh oui, ce n'est pas la fin de l'histoire !).

  • View other solutions - lorsque le challenge est résolu, vous pouvez accéder aux solutions des autres joueurs qui sont répartis en plusieurs catégories.
  • Publish your solution - publier votre propre solution.
  • Next Mission - aller à la mission suivante.

Je vous recommande de parcourir les solutions des autres joueurs avant de publier la vôtre.
