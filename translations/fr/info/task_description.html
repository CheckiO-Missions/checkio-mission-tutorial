Comment ça marche ?

Quand vous commencez à travailler sur une mission, le code initial présente une fonction vide (que vous compléterez avec votre code). Des assertions sont incluses dans le code pour vérifier que la fonction se comporte comme prévu. Assurez-vous que la fonction renvoie les valeurs demandées, elle ne les affiche pas. {% if interpreter.slug == "js-node" %} Pour cela, utilisez la commande "return" au lieu de la fonction "console.log()". {% else %} Pour cela, utilisez la commande "return" au lieu de la fonction "print()". Jetez un œil à cette courte explication. {% endif %}

Les assertions dans le code peuvent être utilisées pour tester votre solution en cliquant sur le bouton "Run Code". CheckiO utilise aussi plusieurs tests additionnels pour vérifier votre solution quand vous cliquez sur le bouton "Check Solution".
