This is an into mission, the purpose of which is to explain how to solve missions on CheckiO and how to get the most out of solving them. When the mission is solved, one more station become available for you, containing more complex missions.

C'est une mission d'échauffement, Le but est d'expliquer comment résoudre les missions sur CheckiO. Quand une mission est résolue, d'autres stations contenant d'autres missions plus difficile seront disponible pour vous.

Cette mission est la plus facile. Ecris une fonction qui recoit deux nombres en entrée et renvoir le résultat de la multiplication de ces deux nombres.

En entrée: Deux arguments. Les deux sont des "integers".

En sortie: Un "integer".

Example:

{% if interpreter.slug == "js-node" %} 
multTwo(2, 3) == 6
multTwo(1, 0) == 0
{% else %} 
mult_two(2, 3) == 6
mult_two(1, 0) == 0
{% endif %}

Commment tout celà fonctionne ?

Au départ, le code est une fonction vide (que vous allez avoir à remplir). Ta fonction doit retourner des valeurs et surtout pas les afficher à l'écran. {% if interpreter.slug == "js-node" %} That is, use the return command instead of the console.log function. {% else %} That is, use the return command instead of the print function. Check this short explanation. {% endif %}

Tu peux vérifier ta réponse en cliquant sur le boutton "Run Code". CheckiO also uses several additional tests in order to check your solution when you click the “Check Solution” button.

Si ta solution ne passe pas les tests interne, le panel de droite affichera un message d'erreur en trois parties.

  • Fail: - Comment ta fonction a été appellée.
  • Your Result: - Ce qui à été retourné.
  • Right Result: - Ce qui est attendu.

Pour résoudre ce challenge, la fonction vide doit être remplacée par le code suivant :

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Clique sur le boutton "check".

Si ta solution passe tous les tests, des messages de félicitations apparaitront sur le panneau de droite suivi d'une suggestion pour la suite (Oui, ce n'est jamais la fin de l'histoire).
