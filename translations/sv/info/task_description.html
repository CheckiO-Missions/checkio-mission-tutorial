Det här är ett introduktionsuppdrag med syfte att förklara hur du utför uppdrag här på CheckiO och hur du får ut så mycket som möjligt av att lösa dem. När du har utfört uppdraget får du tillgång till en ny station med mer avancerade uppdrag.

Det här är därmed det lättaste uppdraget. Skriv en funktion som tar två heltal som indata och multiplicerar dem.

Indata: Två argument. Båda är heltal.

Utdata: Ett heltal.

Exempel:

{% if interpreter.slug == "js-node" %} 
multTwo(2, 3) == 6
multTwo(1, 0) == 0
{% else %} 
mult_two(2, 3) == 6
mult_two(1, 0) == 0
{% endif %}

Hur fungerar det?

När du påbörjar en uppgift består koden alltid av en "tom" funktion (där du sedan ska skriva din kod) och ett antal testfall som kontrollerar att din lösning är korrekt. När du skriver din funktion är det viktigt att du returnerar resultatet och inte bara skriver ut det. {% if interpreter.slug == "js-node" %} Med andra ord behöver du använda return-kommandot istället för funktionen console.log. {% else %} Med andra ord behöver du använda return-kommandot istället för funktionen print. Förklaring {% endif %}

Testfallen efter din funktion kan användas för att kontrollera din lösning innan du skickar in den. För att göra detta, klicka på "Run Code"-knappen. När du sedan klickar på "Check Solution"-knappen kommer CheckiO att köra ytterligare testfall för att kontrollera att din lösning är korrekt.

Om din lösning inte klarade testfallen så kommer panelen till höger att visa ett felmeddelande innehållande följande tre punkter:

  • Fail: - visar hur din funktion anropades.
  • Your Result: - visar resultatet av din funktion.
  • Right Result: - visar det rätta resultatet.

För att lösa den här uppgiften måste den "tomma" funktionen bytas ut mot följande:

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Prova sedan att klicka på "Check"-knappen.

Om din lösning klarade alla testfallen kommer panelen till höger att visa en gratulation följd av förslag på vad du kan göra härnäst. (Nej, din resa slutar inte här).

  • View other solutions - när du har löst uppgiften kan du se hur andra spelare har löst den. Lösningarna är indelade i olika kategorier.
  • Publish your solution - publicera din egen lösning så att andra spelare kan se och kommentera den.
  • Next Mission - fortsätt till nästa uppdrag.

Jag rekommenderar att du går igenom andra spelares lösningar innan du publicerar din egen.

Sist men inte minst så har vissa uppdrag en lista med ledtrådar som du kan läsa vid behov, men eftersom vi redan har berättat hur du löser den här uppgiften så är "ledtrådarna" i detta fallet några intressanta fakta om CheckiO.
