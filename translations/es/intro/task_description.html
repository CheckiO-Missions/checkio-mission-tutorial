Esta es una misión introductoria, cuyo objetivo es explicar cómo completar misiones en CheckiO y cómo aprovechar al máximo su resolución. Una vez completada la misión, se habilitará otra estación que contiene misiones más complejas

Esta misión es la más sencilla: Escribir una función que recibirá dos números como Entrada y deberá retornar la multiplicación de esos dos números como Salida.

Entrada: Dos argumentos, ambos de tipo int.

Salida: Int.

Ejemplo:

{% if interpreter.slug == "js-node" %} 
multTwo(2, 3) == 6
multTwo(1, 0) == 0
{% else %} 
mult_two(2, 3) == 6
mult_two(1, 0) == 0
{% endif %}

¿Cómo funciona?:

Cuando comienzas a trabajar en el problema, verás que el código inicial consiste en una función “vacía” (que deberás completar con tu código). A su vez, vienen incluídas aserciones para verificar que la función tenga el comportamiento esperado. Debes tomar en cuenta que la función retorna valores y no los imprime. {% if interpreter.slug == "js-node" %} Para realizar esto, usa el comando 'return' en lugar de la función 'console.log'. {% else %} Para realizar esto, usa el comando 'return' en lugar de la función 'print'. Revisa esta breve explicación. {% endif %}

Las aserciones en el código pueden ser usadas para chequear tu solución al presionar el botón “Run”(). CheckiO también ejecuta varios test adicionales para chequear tu solución cuando haces clic en el botón “Check”().

Si la solución que escribiste no pasa los tests internos, el panel derecho mostrará un mensaje de error compuesto por 3 ítems:

  • Your Result: - Muestra lo que retornó.
  • Right Result: - Muestra lo que debió haber retornado.
  • Fail: - Muestra cómo fue invocada tu función.

Para resolver la actividad, el contenido de la función vacía debe ser reemplazado con el siguiente código:

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Ahora intenta hacer clic en el botón “Check”.

Si la solución pasa todos los tests, deberán aparecer las felicitaciones en el panel derecho, junto a las siguientes sugerencias (sí, este no es el final de la historia):

  • View other solutions - Una vez que la actividad ha sido completada, podrás acceder a la solución realizada por otros jugadores, las cuales están divididas por categorías.
  • Publish your solution - Publica tu solución personal.
  • Next Mission - Avanza a la siguiente misión.

Es recomendable revisar las soluciones de otros jugadores antes de publicar tu solución personal.

Por último, pero no menos importante, algunas actividades tienen al final una lista de pistas de resolución, pero como en esta ya hemos descrito cómo resolverla, en su lugar, agregaremos datos interesantes acerca de CheckiO.
