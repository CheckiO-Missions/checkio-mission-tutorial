Представьте, что вам нужно переменожить два числа и вывести результат на экран. Например, 5 и 6.

print(5*6)

Результат:

30

Теперь вместо 5 и 6 мы будем использовать переменные. Зачем? Увидите в следующих примерах.

a = 5
b = 6
print(a*b)

Результат:

30

Вы можете изменить переменную a, перезапустить скрипт – и получите другой результат.

a = 6
b = 6
print(a*b)

Результат:

36

Или скопировать код дважды, изменить переменные и увидеть результаты двух произведений.

a = 5
b = 6
print(a*b)


a = 6
b = 6
print(a*b)

Результат:

30
36

Сейчас я покажу, как с помощью функции можно упростить этот код.

def mult_two(a, b):

    return a*b

print(mult_two(5, 6))
print(mult_two(6, 6))

Результат:

30
36

Как видите, функция mult_two теперь содержит всю логику, и достаточно лишь поменять её аргументы, чтобы получить нужный результат.

Конечно. На этой картинке показана структура функции:

А вот ещё раз наша функция:

def mult_two(a, b):

    return a*b

Она всегда начинается с def.

Функция называется mult_two - это имя можно использвать, чтобы в будущем вызывать функцию

Два аргумента (a, b) - это переменные, которые получают свои значения при вызове функции. Например, когда мы вызываем функцию mult_two(5, 6) - внутри функции перменной a присваивается значение 5, а переменной b - 6.

Также можно задать функцию с указанием типов (type hints).

def mult_two(a: int, b: int) -> int:

    return a*b

Пусть вас не смущают выражения : int и -> int. Они поясняют пользователю (тому, кто читает код) ожидаемые типы аргументов и тип результата, который возвращает функция. Но пока эти типы вполне можно игнорировать.

Далее, строка внутри функции, которая начинается с четырёх пробелов и слова return говорит, что получится в результате вызова функции return a*b.

В нашем примере print(mult_two(5, 6)) - вернётся результат перемножения двух чисел 5 и 6, который затем будет передан как аргумент в другую функцию print.

Если в функции более одной строки кода, все они должны начинаться с четырёх пробелов. Например, добавим ещё одну переменную c, в которую мы присвоим результат произведения и вернём как результат функции

def mult_two(a, b):

    c = a*b
    
    return c

Вверху редактора кода есть две кнопки: “Run Code” и “Check Solution”.

Когда нажимаете первую - она запускает код, написанный в редакторе. Когда нажимаете “check solution” - система проверяет вашу функцию и показывает поздравление, если она работает как положено.

Если что-то не работает, на экране покажется вызов функции, для которого вернулось неправильное значение

или когда ошибка произошла в assert-проверке
