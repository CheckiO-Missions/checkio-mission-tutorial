To jest misja wprowadzająca, której celem jest wyjaśnienie jak rozwiązywać misje CheckiO. Gdy misja zostanie ukończona, uzyskasz dostęp do kolejnej stacji zawierającej więcej misji o wyższym poziomie trudności.

Ta misja jest najłatwiejsza. Napisz funkcję, która na wejściu przyjmuje 2 liczby i zwraca ich iloczyn.

Wejście: Dwa argumenty. Oba typu int.

Wyjście: Int.

Przykład:

{% if interpreter.slug == "js-node" %} 
multTwo(2, 3) == 6
multTwo(1, 0) == 0
{% else %} 
mult_two(2, 3) == 6
mult_two(1, 0) == 0
{% endif %}

Jak to działa?:

Gdy zaczynasz rozwiązywać misję, kod początkowo zawiera “pustą” funkcję (którą musisz uzupełnić swoim rozwiązaniem), a poniżej znajdują się asserty. Powinieneś zwrócić uwagę na to, że twoja funkcja powinna zwracać wartości, a nie je wyświetlać. {% if interpreter.slug == "js-node" %} W tym celu używaj komendy return zamiast funkcji console.log. {% else %} W tym celu używaj komendy return zamiast funkcji print. Dowiedz się więcej w tym krótkim wyjaśnieniu. {% endif %}

Możesz użyć assertów znajdujących się poniżej twojej funkcji w celu sprawdzenia swojego rozwiązania wciskając przycisk “Run” (). CheckiO używa też kilku dodatkowych testów aby sprawdzić twoje rozwiązanie po naciśnięciu przycisku “Check” ().

Jeśli rozwiązanie nie przejdzie testów wewnętrznych, panel po prawej stronie wyświetli informację o błędzie zawierającą 3 pozycje.

  • Fail: - jak twoja funkcja została wywołana.
  • Your Result: - co zwróciła.
  • Right Result: - co powinna zwrócić.

Aby rozwiązać zadanie, “pusta” funkcja musi zostać zastąpiona następującym kodem:

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Spróbuj teraz nacisnąć przycisk “Check”.

Jeśli rozwiązanie przejdzie wszystkie testy, na prawym panelu powinny pojawić się gratulacje wraz z sugestią dotyczącą dalszego działania. (Tak, to nie koniec historii).

  • View other solutions - pokaż inne rozwiązania. Po rozwiązaniu zadania możesz uzyskać dostęp do podzielonych na kategorie rozwiązań innych użytkowników.
  • Publish your solution - udostępnij swoje rozwiązanie.
  • Next Mission - idź do kolejnej misji.

Przed opublikowaniem własnego rozwiązania polecamy zapoznanie się z rozwiązaniami innych użytkowników.
